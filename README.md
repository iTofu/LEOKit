## Installation

### CocoaPods

LEONetworkKit is available on [CocoaPods](https://cocoapods.org/). Just add the following to your project Podfile:

```ruby
pod 'LEONetworkKit' # Podfile
```

### Non-CocoaPods

Just drag the LEONetworkKit folder into your project.
